Project Title: Build Automation using Maven

Objective:
To automate the build process of a Java application using Apache Maven.

Tools Used:

* Apache Maven
* Java (JDK 17)
* Git Bash / Command Prompt

Project Description:
This project demonstrates how Maven is used to automate the build lifecycle of a Java application. It includes dependency management, compilation, packaging, and generation of a JAR file.

Steps Performed:

1. Installed Java and Maven
2. Created Maven project using archetype
3. Modified pom.xml and added dependency
4. Built project using "mvn clean install"
5. Verified output in target folder

Output:
A JAR file (build-automation-1.0-SNAPSHOT.jar) is generated successfully in the target directory.

Result:
The build process was successfully automated using Maven.
